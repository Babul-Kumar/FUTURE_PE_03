# AI SEO Content Cluster Generator

## Overview
This project demonstrates how prompt engineering can be used to create
SEO-optimized content clusters for business websites.

## Business Chosen
SmileCare Dental Clinic  
Location: Bangalore  
Type: Dental Clinic

## Objective
To build a reusable AI-driven SEO system that generates:
- Pillar blogs
- Supporting cluster blogs
- Local SEO-focused content
- Lead-generating CTAs

## Tools Used
- ChatGPT
- Google Search (keyword inspiration)

## Folder Structure
- prompts/: SEO prompt templates
- outputs/: Generated SEO blog content
- README.md: Project documentation

## Outcome
The content produced is:
- Search intent focused
- SEO-optimized
- Ready for real business websites

This system can be reused for clinics, agencies, coaching institutes, and SaaS businesses.
